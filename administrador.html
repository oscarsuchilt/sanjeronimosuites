📋 SQL para crear tabla en Supabase

Pega esto en SQL Editor de tu proyecto Supabase:

CREATE TABLE suites (
  id TEXT PRIMARY KEY,
  nombre TEXT,
  tipo TEXT,
  piso INTEGER,
  descripcion TEXT,
  tags TEXT[],
  whatsapp TEXT,
  airbnb_url TEXT,
  fotos TEXT[],
  orden INTEGER DEFAULT 0,
  activa BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Crear bucket "fotos" en Storage → New bucket → "fotos" → Public
